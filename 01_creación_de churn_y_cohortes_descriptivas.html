3. Definici√≥n de Variable Target

Para los fines de este estudio, se define la variable churn como la situaci√≥n en la que una persona ha sido diagnosticada con una condici√≥n de salud mental (Mental_Health_Condition = Yes) pero no ha tenido consultas m√©dicas registradas (Consultation_History = No). En este caso, churn = 1. El resto de los individuos se codifican como churn = 0.

# Funci√≥n para calcular el intervalo de confianza de Wilson
def wilson_ci(successes, n, alpha=0.05):
    ci_low, ci_high = proportion_confint(successes, n, alpha=alpha, method="wilson")
    return ci_low, ci_high

  • El intervalo de confianza de Wilson (IC95%: 24.74% ‚Äì 25.50%) nos da un rango de valores plausibles para la tasa real de churn en la poblaci√≥n, con un 95% de confianza estad√≠stica.Es decir, si repiti√©semos este estudio en m√∫ltiples muestras de la misma poblaci√≥n, el 95% de esos intervalos construidos contendr√≠an la tasa verdadera de abandono.

5. Cohortes descriptivas

De acuerdo con los hallazgos del EDA inicial (Notebook 0), se tomaron las siguientes decisiones metodol√≥gicas para garantizar coherencia y robustez en el an√°lisis:

  1. Valores nulos

    • La variable Severity presentaba ~50% de valores ausentes.
    • En lugar de eliminar casos, se cre√≥ una categor√≠a expl√≠cita Unknown para mantener el tama√±o muestral y analizar este grupo como un perfil de riesgo separado.

  2. Outliers

    • Con el criterio cl√°sico de IQR no se detectaron valores at√≠picos en las variables num√©ricas (Age, Sleep_Hours, Work_Hours, Physical_Activity_Hours, Social_Media_Usage).
    • Por ello, no fue necesario excluir ni transformar observaciones extremas.

  3. Normalidad de variables num√©ricas

    • Las pruebas de D‚ÄôAgostino K¬≤ rechazaron normalidad en todas las variables num√©ricas (p ‚âà 0).
    • Esto justific√≥ el uso de discretizaci√≥n en intervalos (binning) para facilitar la construcci√≥n de cohortes y la aplicaci√≥n de pruebas no param√©tricas.

  4. Construcci√≥n de cohortes

    • Se agruparon las variables num√©ricas en intervalos te√≥ricamente relevantes:
      • Age_group ‚Üí (18‚Äì29, 30‚Äì44, 45‚Äì59, 60+).
      • Sleep_group ‚Üí (<5h, 5‚Äì7h, 7‚Äì9h, >9h).
      • Work_group ‚Üí (‚â§40h, 41‚Äì55h, 56‚Äì70h, >70h).
      • PA_group ‚Üí (<2h, 2‚Äì5h, 5‚Äì8h, >8h).
      • SM_group ‚Üí (‚â§1h, 1‚Äì3h, 3‚Äì5h, >5h).

def bin_hours(s, bins, labels):
    return pd.cut(s, bins=bins, labels=labels, include_lowest=True)

def wilson_ci(k, n, alpha=0.05):
    lo, hi = proportion_confint(k, n, alpha=alpha, method="wilson")
    return lo, hi

def plot_cohort_with_ci(tbl, var_name, fname):
    """
    Espera una tabla con columnas: [var_name, n, churn_1, rate, ci_low, ci_high]
    Crea un barplot con barras de error (IC95%) y guarda en fig/.
    """
    # Orden por tasa
    tbl = tbl.sort_values("rate", ascending=False).reset_index(drop=True)
    # Error bars en puntos porcentuales
    err_low = (tbl["rate"] - tbl["ci_low"]) * 100
    err_high = (tbl["ci_high"] - tbl["rate"]) * 100
    err = [err_low.values, err_high.values]
    
    plt.figure(figsize=(7,4))
    ax = sns.barplot(x=var_name, y=tbl["rate"]*100, data=tbl, edgecolor="black", palette="Set2")
    # A√±adir barras de error manualmente
    ax.errorbar(x=range(len(tbl)), y=tbl["rate"]*100, 
                yerr=err, fmt='none', capsize=3, lw=1.2, color="black")

    plt.ylabel("Churn (%)")
    plt.xlabel(var_name)
    plt.xticks(rotation=45, ha="right")
    plt.title(f"Churn por {var_name} (IC95%)")
    plt.tight_layout()
    out_path = f"fig/{fname}"
    plt.savefig(out_path, bbox_inches="tight", dpi=150)
    plt.show()
    print("Guardado:", out_path)

# An√°lisis de Chi-cuadrado y Cram√©r's V para cada cohorte vs churn
def cramers_v(tab):
    """Cram√©r's V sin correcci√≥n de sesgo (suficiente como m√©trica de intensidad)."""
    chi2, p, dof, exp = chi2_contingency(tab)
    n = tab.values.sum()
    r, c = tab.shape
    return np.sqrt(chi2 / (n * (min(r-1, c-1) or 1))), p, dof

results = []
for var in cohort_vars:
    # Tabla de contingencia cohorte √ó churn
    tab = pd.crosstab(df[var], df["churn"])
    v, p, dof = cramers_v(tab)
    # Guardar tabla y resultado
    tab.to_csv(f"out/xtab_{var}_by_churn.csv")
    results.append({"variable": var,
                    "n": int(tab.values.sum()),
                    "levels": int(tab.shape[0]),
                    "chi2_pvalue": p,
                    "cramers_v": v,
                    "dof": int(dof)})

res_df = pd.DataFrame(results).sort_values(["cramers_v","chi2_pvalue"], ascending=[False, True])

# Funci√≥n para dibujar heatmap de tasa de churn
def heatmap_churn_rate(var, min_cell=100, fname=None):
    """
    Dibuja heatmap de tasa de churn por categor√≠a de var.
    Enmascara categor√≠as con n < min_cell.
    """
    # Tabla resumida (n y tasa)
    g = df.groupby(var)["churn"].agg(n="count", churn_1="sum", rate="mean").reset_index()
    g["rate_pct"] = 100 * g["rate"]
    g["mask"] = g["n"] < min_cell

    # Orden por tasa
    g = g.sort_values("rate_pct", ascending=False)

    # Heatmap 1D (categor√≠as en eje Y)
    plt.figure(figsize=(6, max(3, 0.35*len(g))))
    # Creamos una matriz 2D (rate) para usar heatmap f√°cilmente
    mat = g[["rate_pct"]].values
    ax = sns.heatmap(mat,
                     annot=g[["rate_pct"]].round(2).values,
                     fmt=".2f",
                     cmap="coolwarm",
                     cbar=True,
                     yticklabels=g[var].astype(str).tolist(),
                     xticklabels=["Churn (%)"],
                     vmin=g["rate_pct"].min(),
                     vmax=g["rate_pct"].max())
    # Enmascarado visual: cruzamos las celdas de baja frecuencia
    for i, low in enumerate(g["mask"].values):
        if low:
            ax.add_patch(plt.Rectangle((0, i), 1, 1, fill=False, edgecolor="red", linewidth=2, linestyle="--"))
    plt.title(f"Tasa de churn por {var} (rojo = n < {min_cell})")
    plt.tight_layout()
    outp = f"fig/heatmap_churn_{var}.png" if fname is None else f"fig/{fname}"
    plt.savefig(outp, dpi=150, bbox_inches="tight")
    plt.show()
    print("Guardado:", outp)
