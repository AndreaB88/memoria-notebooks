def memory_reduce(df: pd.DataFrame) -> pd.DataFrame:
    for col in df.select_dtypes(include=["int64","int32","int16"]).columns:
        df[col] = pd.to_numeric(df[col], downcast="integer")
    for col in df.select_dtypes(include=["float64","float32"]).columns:
        df[col] = pd.to_numeric(df[col], downcast="float")
    return df

def to_category(df, cols):
    for c in cols:
        if c in df.columns:
            df[c] = df[c].astype("category")
    return df

def iqr_outlier_mask(s: pd.Series, k: float = 1.5):
    q1, q3 = s.quantile([0.25, 0.75])
    iqr = q3 - q1
    lower, upper = q1 - k*iqr, q3 + k*iqr
    return (s < lower) | (s > upper)

def wilson_ci(k, n, z=1.96):
    if n == 0: return (np.nan, np.nan)
    p = k/n
    denom = 1 + z**2/n
    center = (p + z**2/(2*n))/denom
    margin = (z*math.sqrt((p*(1-p) + z**2/(4*n))/n))/denom
    return center - margin, center + margin

def cramers_v(table: pd.DataFrame):
    chi2, p, dof, exp = chi2_contingency(table)
    n = table.values.sum()
    r, c = table.shape
    return math.sqrt(chi2 / (n * (min(r-1, c-1) or 1))), p

def savefig(path):
    plt.tight_layout()
    plt.savefig(path, bbox_inches="tight", dpi=150)
    plt.show()

# Agrupación de categorías
def topN_other(series, top=8):
    vc = series.value_counts(dropna=False)
    keep = vc.index[:top]
    return series.where(series.isin(keep), other="Other").astype("category")

cat_summary = []
for c in cat_cols:
    # reagrupa si hay demasiadas categorías
    if df[c].nunique() > 12:

1. Correlaciones entre variables numéricas (Spearman)¶

El análisis de correlación de Spearman entre las variables continuas (Age, Sleep_Hours, Work_Hours, Physical_Activity_Hours, Social_Media_Usage) mostró coeficientes cercanos a cero en todos los casos.

  • No se observa dependencia lineal ni monótona entre las variables numéricas. Cada dimensión parece comportarse de forma independiente.
  • La baja correlación implica que no existe multicolinealidad fuerte, lo que facilita el uso de estas variables como predictores en modelos de regresión o clustering.
  • Aunque en la realidad esperaríamos que ciertas prácticas se relacionaran (ej. más horas de trabajo con menos sueño o actividad física), la independencia encontrada refuerza la idea de tratar cada comportamiento como un eje autónomo dentro del análisis de perfiles.

2. Asociaciones Numéricas Categóricas (Kruskal-Wallis)¶

  • Work_Hours ↔ Medication_Usage (p=0.003): fuerte asociación; mayor carga laboral se relaciona con uso de medicación.
  • Sleep_Hours ↔ Stress_Level (p=0.02): niveles de estrés afectan el sueño.
  • Efectos marginales: ocupación con sueño, estrés con actividad física.
  • Se observan vínculos claros entre estrés, trabajo y salud, consistentes con teorías de precarización laboral y desgaste psicológico.

3. Asociaciones Categóricas Categóricas (Chi² + V de Cramér)¶

  • Gender ↔ Alcohol_Consumption (p=0.024): asociación débil, pero significativa; el consumo responde también a normas de género.

  • Country ↔ Medication_Usage (p=0.046): leves diferencias según país, posiblemente ligadas a acceso desigual a salud.

  • Consultation_History ↔ Stress_Level (p=0.057): tendencia a que quienes reportan más estrés busquen más consultas médicas.

  • las categorías muestran patrones culturales y estructurales, como diferencias en acceso a tratamientos por país o normas sociales de género en consumo de alcohol.

  • No existen correlaciones fuertes entre numéricas, pero sí asociaciones significativas en pares num–cat y cat–cat. Los tamaños de efecto son bajos, lo que refleja fenómenos multicausales.

  • La independencia relativa de las variables numéricas permite usarlas como insumos en clustering y modelos explicativos. Las asociaciones detectadas justifican la inclusión de variables como estrés, medicación y consultas como predictores relevantes en la construcción de cohortes de riesgo.

  • Se confirma que los factores laborales, de estrés, género y acceso a salud emergen como dimensiones estructurales en la explicación del bienestar y riesgo de abandono. Esto aporta sentido social y contextual al análisis, más allá de lo meramente estadístico.

  • La base de datos ofrece una estructura clara y balanceada, adecuada para pruebas de clasificación y segmentación.
  • El rechazo de normalidad en variables numéricas refuerza la necesidad de métodos no paramétricos o modelos que no asuman distribución normal.
  • La ausencia de outliers facilita la comparabilidad
  • La variable Severity requerirá un tratamiento especial en imputación o exclusión, dado su nivel crítico de valores ausentes.

El EDA inicial muestra que los datos son consistentes, completos en su mayoría y con distribuciones balanceadas. Existen algunas asociaciones significativas que permiten plantear hipótesis en torno a los efectos del estrés, las horas de trabajo y el uso de medicación en la salud mental.
